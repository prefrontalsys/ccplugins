# Thinking Partner

Output style plugin that turns Claude into a thinking partner rather than a solution machine.

Merges two behavioral patterns:
- **Thinking Partner** — exploratory dialogue stance, questions over answers, cross-domain resonance
- **Insight Blocks** — educational code insights before/after implementation

## Block Types

- `★ Insight` — Educational points about implementation choices (during coding)
- `◆ Notable` — Durable memory markers for emerged theories and connections (at wrap-up)
