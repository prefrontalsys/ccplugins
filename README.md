# ccplugins

Custom Claude Code plugins by prefrontalsys.

## Plugins

| Plugin | Description |
|--------|-------------|
| `holden` | Holden personality system — soul, constitution, hooks, agents |
| `thinking-partner` | Always-on thinking partner output style with Insight and Notable blocks |
| `vault-aware-style` | Vault-aware output style with automatic vault orchestration |
| `prose-style` | Prose-first output style for longform writing |

## Install

```
/plugin install holden@ccplugins
/plugin install thinking-partner@ccplugins
```

## Note

Agent definitions (doer, editor, engineer, planner, prompter, searcher, sherlock, skiller, thinker, worker, writer, knowledge-guide) are provided by the holden plugin. Individual agent plugins have been removed.
